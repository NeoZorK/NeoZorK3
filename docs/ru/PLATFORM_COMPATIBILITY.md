# Обзор совместимости платформ для NeoZorK3 Arbitrage Bot

## Обзор

NeoZorK3 Solana Arbitrage Bot разработан для истинной кроссплатформенности, поддерживая основные операционные системы и среды сборки. Этот документ предоставляет обзор совместимости платформ и ссылки на подробную документацию.

## Поддерживаемые платформы

### ✅ Linux (Ubuntu/Debian/CentOS)
- **C библиотека**: glibc
- **Компилятор**: GCC/Clang
- **Менеджер пакетов**: apt/yum/dnf
- **Статус**: ✅ ПОЛНОСТЬЮ ПОДДЕРЖИВАЕТСЯ
- **Документация**: [Совместимость с Linux](../en/LINUX_COMPATIBILITY.md)

### ✅ Alpine Linux
- **C библиотека**: musl libc
- **Компилятор**: GCC
- **Менеджер пакетов**: apk
- **Статус**: ✅ ПОЛНОСТЬЮ ПОДДЕРЖИВАЕТСЯ
- **Документация**: [Совместимость с Alpine](../en/ALPINE_COMPATIBILITY.md)

### ✅ Windows
- **C библиотека**: MSVCRT
- **Компилятор**: MSVC (Visual Studio)
- **Менеджер пакетов**: vcpkg
- **Статус**: ✅ ПОЛНОСТЬЮ ПОДДЕРЖИВАЕТСЯ
- **Документация**: [Совместимость с Windows](../en/WINDOWS_COMPATIBILITY.md)

### ✅ macOS
- **C библиотека**: libSystem
- **Компилятор**: Clang

- **Статус**: ✅ ПОЛНОСТЬЮ ПОДДЕРЖИВАЕТСЯ
- **Документация**: [Инструкции по сборке](../en/BUILD_INSTRUCTIONS.md)

## Быстрый старт по платформам

### Linux (Ubuntu/Debian)
```bash
# Установка зависимостей
sudo apt-get update
sudo apt-get install -y build-essential cmake libboost-all-dev libssl-dev libgtest-dev git nlohmann-json3-dev

# Сборка
./build.sh
```

### Alpine Linux
```bash
# Установка зависимостей
apk update
apk add build-base cmake boost-dev openssl-dev gtest-dev git nlohmann-json

# Сборка
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)
```

### Windows
```cmd
# Установка vcpkg и зависимостей
git clone https://github.com/Microsoft/vcpkg.git C:\vcpkg
cd C:\vcpkg
.\bootstrap-vcpkg.bat
.\vcpkg integrate install
.\vcpkg install boost-system:x64-windows boost-thread:x64-windows boost-beast:x64-windows openssl:x64-windows nlohmann-json:x64-windows

# Сборка (из Developer Command Prompt)
build_windows.bat
```

### macOS
```bash
# Установка зависимостей
brew install cmake boost openssl googletest nlohmann-json

# Сборка
./build.sh
```

## Система сборки

Все платформы используют **CMake** как основную систему сборки:

